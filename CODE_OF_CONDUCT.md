# Кодекс поведения

## Наш принцип
Проект открыт для людей и AI-агентов. Мы за уважительное, по существу и доброжелательное общение — как принято в
здоровом инженерном сообществе 1С.

## Ожидаемое
- Уважение к участникам вне зависимости от опыта, взглядов, языка.
- Конструктивная критика идей и кода, а не людей.
- Терпимость к вопросам новичков; помощь вместо снисходительности.

## Недопустимо
- Оскорбления, троллинг, харассмент, дискриминация.
- Публикация чужих персональных данных.
- Намеренный вред проекту (вредоносный код, спам).

## Применение
Нарушения сообщайте мейнтейнеру приватно (через профиль владельца репозитория или GitHub Security Advisories для
чувствительных случаев). Мейнтейнер вправе редактировать/удалять неприемлемый контент и ограничивать участие.

Основано на духе [Contributor Covenant](https://www.contributor-covenant.org/) в сокращённой форме.
